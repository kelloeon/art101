Lab 7 - Functions

Challenge

The goal of the lab was to create a JavaScript program that scrambles a string that the user inputs when prompted and outputs it. The prompt is the user's name.

Problems

I had two major problems while coding for this lab. One of them was that my website was prompting the user twice every time it opened, which I was able to solve after finding out from a friend that the script was being called twice in my HTML: once at the head of the document and again in my "Results" section. I grayed out the first time the script was called and it resolved the issue of the window being called twice. The other problem I ran into was that my JavaScript would not show up at all when opening my site. It would prompt the user to enter their name, then nothing would show up in the "Results" section. I eventually found that it was a syntax error: I had written "let nameArraySort = nameArray.sort("");" instead of leaving the parentheses empty. I had no idea that such a minor syntax error could cause such a major problem, but I'm super relieved to have found it.
